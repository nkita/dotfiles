# 機能一覧

## Neovim

### ファイルパスのコピー

| キー | 動作 |
|---|---|
| `<leader>yf` | ファイル名をコピー |
| `<leader>yp` | 相対パスをコピー |
| `<leader>ya` | 絶対パスをコピー |

### コメントアウト

| キー | 動作 |
|---|---|
| `Ctrl+/` | 行/選択範囲をコメントアウト |

### Git管理（LazyGit）

| キー | 動作 |
|---|---|
| `<C-t>` | LazyGit起動 |
| `<leader>gf` | 現在のファイルの履歴をLazyGitで表示 |

### LSP診断とナビゲーション

| キー | 動作 |
|---|---|
| `gl` | カーソル行の診断を表示 |
| `<leader>fd` | 診断一覧（Telescope） |
| `<leader>fD` | 現在のファイルの診断 |
| `<leader>q` | 診断をQuickfixに表示 |
| `[d` / `]d` | 前/次の診断へジャンプ |
| `gd` | 定義へジャンプ |
| `gD` | 宣言へジャンプ |
| `gi` | 実装へジャンプ |
| `gr` | 参照一覧 |
| `K` | ホバー情報を表示 |

### ファイル検索・Grep（Telescope）

| キー | 動作 |
|---|---|
| `<leader>f` | ファイル名で検索 |
| `<leader>r` | 内容をgrep検索 |
| `Ctrl+E` | ファイルツリー表示 |

### Markdownプレビュー

| キー | 動作 |
|---|---|
| `<leader>md` | Glowプレビュー |
| `<leader>mp` | Glowプレビュー |

### Node.jsデバッグ

| キー | 動作 |
|---|---|
| `F5` | デバッグ開始/継続 |
| `F1` | ステップイン |
| `F2` | ステップオーバー |
| `F3` | ステップアウト |
| `F7` | デバッグUIの表示/非表示 |
| `<leader>b` | ブレークポイント切り替え |

---

## zsh カスタム関数

| コマンド | 動作 |
|---|---|
| `ff [dir]` | ファイル検索してNeovimで開く（プレビュー付き） |
| `gf [dir]` | grep検索してNeovimで該当行を開く（プレビュー付き） |
| `gwa <branch>` | git worktreeを追加して移動 |
| `gwr [branch]` | git worktreeを削除（現在地なら引数省略可） |

`ff` / `gf` ともに `Ctrl+/` でプレビューの表示/非表示を切り替えられます。
